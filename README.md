# AKmart-Sales-Analysis-Dashboard-Power-BI-
Interactive Power BI dashboard analyzing AKmart sales, product performance, customer trends, outlet insights, and key business metrics for decisions.

>>>**AKmart Sales Analysis Dashboard | Power BI**

>>>📌 **Project Overview**
This project showcases an interactive Power BI dashboard built to analyze AKmart's sales performance, customer behavior, and product insights. The dashboard helps identify sales trends, top-performing categories, customer preferences, and key business metrics through interactive visualizations.

>>>📊 **Key Features**
- Total Sales Analysis
- Product Category Performance
- Outlet Type & Size Analysis
- Sales by Location
- Customer Rating Analysis
- Item Type Performance
- Interactive Filters & Slicers
- KPI Cards
- DAX Measures
- Business Insights Dashboard

>>>< 🛠 **Tools Used**
- Power BI
- Power Query
- DAX
- Data Modeling
- Excel
>>> 📈 **Dashboard Insights**
- Analyze total sales and revenue performance.
- Identify top-selling product categories.
- Compare sales across outlet types and locations.
- Monitor customer ratings and purchasing trends.
- Support business decisions using interactive reports.

>>> 🎯 **Objective**
The objective of this project is to transform Blinkit sales data into meaningful visual insights that help businesses optimize sales strategies, improve customer satisfaction, and drive data-driven decisions.
